更改/更新事件

　　

表单字段的变化

　　在进行表单操作时，你经常会遇到一些在某些选择项被选中，或选择项改变时，去触发某些操作的情况。
而在Chosen里，用户在任何时候选中一个选择项时，都会触发一个change事件。
你可以这样去写这个change操作： 　　
　　$("#form_field").chosen().change( … ); 　　

选择更新动态

　　如果你需要去动态更新select选择框里的选择项，你需要通知Chosen去响应这个变动，
你需要在这个选项框是触发一个"liszt:updated"事件，
这样Chosen就会对更新过内容后的select选择框重新进行渲染。 　　
　　$("#form_field").trigger("chosen:updated"); 　　

自定义宽度支持

　　使用自定义宽度与选择一样容易通过一个选择当您创建选择: 　　
　　 $(".chosen-select").chosen({width: "95%"});
